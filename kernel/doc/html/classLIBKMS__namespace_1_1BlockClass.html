Прототип описания класса блока

Класс блока определяет подпрограмму (subprogram_t), вызываемую вместе с блоком.

Подпрограмма должна добавляться в хранилище подпрограмм автоматически при считывании конфигурации с именем, определенном в классе-наследнике. Подпрограмма возвращает словарь переменных, которые могут использоваться или не использоваться в блоке. Для добавления действующей функции подпрограммы в хранилище использовать метод add_func(...).
